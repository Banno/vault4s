vault4s - Vault Client Library For Scala

Quick Start

To use vault4s in an existing SBT project with Scala 2.11 or a later version, add the following dependencies to your build.sbt depending on your needs:

libraryDependencies ++= Seq(
  "com.banno" %% "vault4s" % "<version>"
)
